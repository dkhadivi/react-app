Deploy to NetlifyReact App

Let's deploy our website to Netlify and host it there.

  1. Login to Netlify. I signed up and logged in.
  2. Select Sites then click Import from Git

  3. Click GitHub to connect to GitHub.

  4. Click the arrow next to dkhadivi.

  5. Select Only select repositories, click Select repositories then dkhadivi/react-app. Click Install at the bottom on the page (not shown).

  6. You're prompted for your GitHub password. Enter it and click Confirm password.

  7. The page closes. The main page shows the selected repository. Click on the repository.

  8. The following page is displayed. The Owner and Brach to deploy are populated automatically. Type build-prod, which is the production build script in the package.json file for production build for Build command. Type dist/ for Publish directory. Click Deploy site.

  9. Wait for the site to deploy.

    10. Netlify adds continuous integration for our deployed site. This means that every time you push to the main branch, Netlify will auto-deploy a new version of the website. It will build the site. If there are any errors, it will not host it, so it won't break your current website. If the build produces no errors, then it will replace the current website with the newly-build one.
